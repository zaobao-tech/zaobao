在 2026 年以前，AI 更像是一个博学的「室友」：你问他问题，他能写代码、写诗，但不能下床帮你倒杯水，更不能帮你把刚写好的代码部署到服务器上。**OpenClaw** 的出现，就是给这个「大脑」装上了手脚，并让它住进了你的微信和飞书。

## 这东西到底是什么？

若不谈晦涩的架构，可以把它看作一个 **「带权限的数字替身」**。

以前我们用 AI，是人类去它的地盘（比如打开 ChatGPT 的网页）；现在用 OpenClaw，是把 AI 请到我们的地盘。它不仅能听懂你说话，最重要的是它有**操作权**：可以读取你丢在飞书里的文档，根据指令抓取网页信息，甚至在你睡觉时按预设逻辑帮你处理杂事。

- **对开发者：** 开源可部署在自己的电脑或私有云上，无需担心隐私泄露。
- **对普通用户：** 复杂配置藏在后台，你面对的仍是熟悉的聊天窗口。

## 如何把它跑起来？（入职教程）

部署 OpenClaw，相当于给「数字管家」办理入职，分三步：

### 1. 挑选「大脑」

你需要为管家选一个「智力来源」（API Key）。

- **国内环境：** 推荐 **DeepSeek** 或 **通义千问**，更懂中文职场里的「潜台词」，响应也快。
- **国际环境：** 推荐 **Claude 3.5 Sonnet**，目前公认逻辑强，处理复杂任务稳定。

### 2. 一键安置

打开终端（Windows 用 PowerShell，Mac 用「终端」），输入：

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

安装完成后执行 `openclaw onboard`，按引导填入 API Key，相当于给管家接通「神经」。

### 3. 连接「通讯机」

在设置里选择要接入的平台：

- **中国用户：** 选「飞书」或「微信」，扫码登录后，AI 就住进你的工作群或私聊里。
- **国际用户：** 选「Telegram」或「WhatsApp」，填入机器人 Token，即成为手机里随叫随到的助手。

## 它能帮你做什么？（场景分享）

- **职场「自动驾驶」：** 你在外面吃饭，老板在群里要上周的财务总结。你在群里 @ 管家：「去我电脑桌面找那几个报销单 Excel，汇总总额发回群里。」它会自己唤醒电脑、找文件、算数并回复，你连筷子都不用放下。
- **24 小时「盯盘员」：** 利用 **Heartbeat（心跳模式）**，可设定：「每隔一小时刷一下小红书上关于『XX 产品』的评论，若有超过 3 个差评，立刻在飞书里弹窗叫我。」
- **跨境「情报官」：** 若你用 Telegram 订阅大量资讯，可让它当过滤器——自动阅读成百上千条推文，只把有价值的内容（如 #半导体 #大模型）精简成几句话发给你。

## 补充与建议

OpenClaw 相比同类工具，一大亮点是对 **MCP（模型上下文协议）** 的支持：AI 能像插 U 盘一样，快速学会使用新工具——例如读本地数据库、操作 Chrome 浏览器。

项目目前更新极快。**国内用户**建议优先选 **DeepSeek** 作为底层模型，处理中文文档和国内网络时，延迟与理解偏差会更小。

不妨先在飞书里拉一个只有你和它的单聊群，试着让它写段代码或查个快递，那种「贾维斯就在手边」的感觉，和网页版完全不同。
